JavaScript Mini Project

This project is a collection of small, interactive JavaScript features that demonstrate DOM manipulation, event handling, and dynamic UI updates. It is designed to showcase core JavaScript logic and beginner-friendly frontend interactions.

Features

- Interactive UI components
- DOM manipulation using JavaScript
- Event handling for buttons, inputs, and user actions
- Responsive and clean layout
- Simple and readable code structure
- Suitable for beginners learning JavaScript fundamentals


Tech Stack

- HTML
- CSS
- JavaScript

Folder Structure

project-folder/
│── index.html
│── style.css
│── script.js
└── assets/ (if any images/icons)

How to Run

1. Download or clone this repository:
https://github.com/Sudhir-dev1718/javascript-mini-project
2. Open the project folder
3. Double-click **index.html** to run it in any browser

No installation required.
